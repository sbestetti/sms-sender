First things first what is Twilio?

Twilio is a platform that offers telecom services in a way that is easy for developers to integrate with applications. These services include SMS, Whatsapp, Voice calls, etc. What they don't offer is a simple interface for people who doesn't know coding to use their services. And that's where this tool comes in. This tools uses your Twilio account to send SMS messages from a simple spreadsheet file. You use your favourite spreadsheet tool (Excel, Google Sheets, etc) to create a simple table with three columns:
  1. The Twilio number that you'll be using to send your messages from
  2. The destination number
  3. The message itself
Add one row for each message you want to send, save the file and use the form at the end of this page to send your messages. That's it! Here's a detailed guide on how to generate the spreadsheet file on the most common tools:

A word about passwords and security

As stated above, this site doesn't store your credentials in any way. But I wouldn't ask you to just take my word for it. That's why I recommend that you use a Token instead of your Twilio password. This might be a bit advanced for most of the users but if you have some understanding of API security, please follow this article to create a token. You can just use a token on the "Password" field instead of your Twilio password.
