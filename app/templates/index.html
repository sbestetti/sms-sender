Bulk SMS Sender

Use your own Twilio account to send SMS messages in bulk from a CSV file

Instructions

Here's the basics: use your favourite spreadsheet software (Excel, Google Sheets, etc) to create a sheet with 3 columns: From, To and Message. Fill it with your data, save it as CSV and upload it to us with your tokens. We'll use Twilio's Messaging API to send your SMS in bulk. Once it's done we'll show you a delivery report.

Here's the detailed steps:

  1. You'll need a Twilio account with a phone number on it for this. Sign up for free at Twilio
  2. Use your favourite spreadsheet software (we recommend Google sheets, but anything that can save a CSV file will work) to create a simple file with 3 columns:
    • From number: your Twilio phone number that will be used to send your messages or an Alpha Sender ID. Check this article for information on how to buy a number on Twilio.
    • To number: the destination number to which the message should be delivered
    • Message: the message itself
  3. Make sure to include the complete number, including the country code. No need to add the plus sign
  4. Once you are finished filling your data, save the file as CSV and upload it using the button below
  5. The file will be checked and we'll list any invalid numbers for you to fix it
  6. We'll use your credentials or tokens to send your messages through Twilio's API. Once it's finished we'll display a delivery report to you
  7. Rinse and repeat
