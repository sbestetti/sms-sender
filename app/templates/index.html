Bulk SMS Sender
First, some important stuff:
  • This site is not related to nor supported or endorsed by Twilio Inc. in any way
  • This site just provides a simple interface for Twilio's SMS service and it's free of charge and supported by ads
  • You'll probably be charged by Twilio though. For the latest info on pricing please check this page
  • This site doesn't store your credentials in any form but it's still safer to use tokens. Please check this page for instructions on how to do it
  • The source code for this site is available on GitHub
How it works: you simply download a sample spreadsheet, fill it with your data, upload it to us with your tokens and we'll use Twilio's Messaging API to send your SMS in bulk. Once it's done we'll show you a delivery report. Here's the detailed steps:
  1. You'll need a Twilio account for this. Sign up for free at Twilio
  2. Download the sample spreadsheet HERE. (Google Sheets integration coming soon)
  3. Fill the spreadsheet with your data (phone numbers, message, etc). Please don't change the sheet's format or it won't work!
  4. Make sure to add all phone numbers on the international format (+15551234)
  5. Once you are finished filling your data upload the file using the button below
  6. The file will be checked and, if everything's alright, you'll be taken to a screen to provide your Twilio credentials. You should really use Tokens! Please check this page for instructions on how to do it
  7. We'll use your tokens to send your messages through Twilio's API. Once it's finished we'll display a delivery report to you
  8. Rinse and repeat
