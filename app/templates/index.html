Bulk SMS Sender

Use your own Twilio account to send SMS messages in bulk from a CSV file

First, some important stuff

  • This site is not related to nor supported or endorsed by Twilio Inc. in any way
  • This site just provides a simple interface for Twilio's SMS service and it's free of charge and supported by ads
  • You'll probably be charged by Twilio though. For the latest info on pricing please check this page
  • This site doesn't store your credentials in any form but it's still safer to use tokens. Please check this page for instructions on how to do it
  • The source code for this site is available on GitHub

Instructions

Here's the basics: use your favourite spreadsheet software (Excel, Google Sheets, etc) to create a sheet with 3 rows: From, To and Message. Fill it with your data, save it as CSV and upload it to us with your tokens. We'll use Twilio's Messaging API to send your SMS in bulk. Once it's done we'll show you a delivery report.

Here's the detailed steps:

  1. You'll need a Twilio account with a phone number on it for this. Sign up for free at Twilio
  2. Use your favourite spreadsheet software to create a simple file with 3 columns:
    • From number: your Twilio phone number that will be used to send your messages or an Alpha Sender ID. Check this article for information on how to buy a number on Twilio.
    • To number: the destination number to which the message should be delivered
    • Message: the message itself
  3. Make sure to add all phone numbers on the international format, with the plus sign (+) in front of them (+15551234...). You might have to add a single quote mark (') in front of the number so it keeps the plus sign
  4. Once you are finished filling your data upload the file using the button below
  5. The file will be checked and, if everything's alright, you'll be taken to a screen to provide your Twilio credentials. You should really use Tokens! Please check this page for instructions on how to do it
  6. We'll use your credentials or tokens to send your messages through Twilio's API. Once it's finished we'll display a delivery report to you
  7. Rinse and repeat
